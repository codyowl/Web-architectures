### urls:

 - Adding company:
   http://127.0.0.1:8000/company/add/

 - Company list out:
   http://127.0.0.1:8000/company/list/

 - Delete a company
   http://127.0.0.1:8000/company/<company_id>/

 - user signup:
   http://127.0.0.1:8000/user/add/

 - Delete an user:
   http://127.0.0.1:8000/user/<company_name>/<user_id>/
